# Face Recognition Attendance System

A face recognition attendance system built using Python and OpenCV.

## Features
- Face detection
- Face recognition
- Attendance tracking

## Technologies Used
- Python
- OpenCV
- NumPy

## Project Structure

```text
Face Reconition/
│
├── recognition.py
├── README.md
└── .gitignore
```

## How to Run

1. Install Python
2. Install OpenCV and NumPy
3. Run:

```bash
python recognition.py
```
